# guangzhou-opc-site

广州类智能体科技有限公司官方网站。

这是一个纯静态 Next.js App Router 单页展示网站，面向高校毕业生 AI 创业与职业发展 OPC 生态平台。

## Tech Stack

- Next.js App Router
- React
- JavaScript
- Three.js
- @react-three/fiber
- @react-three/drei
- Framer Motion
- Lucide React
- Global CSS

## Local Development

```bash
npm install
npm run dev
```

Local URL:

```text
http://localhost:3000
```

## Build

```bash
npm run lint
npm run build
```

## Notes

- No database
- No API routes
- No login or authentication
- No form submission
